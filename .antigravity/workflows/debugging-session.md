# Debugging Session Workflow

## Paso 1: Reproducir
Definir el síntoma y cuándo ocurre.

## Paso 2: Delimitar
Identificar archivos, estado y rutas involucradas.

## Paso 3: Hipótesis
Generar causas probables.

## Paso 4: Verificación
Comprobar cada hipótesis con razonamiento o cambios controlados.

## Paso 5: Corrección
Aplicar la solución más segura y coherente.

## Paso 6: Validación
Confirmar que el bug quedó resuelto y no generó efectos secundarios.
