# Automatic Challenge Generation

## Objetivo
Generar retos automáticamente usando:
- análisis del repositorio,
- debilidades,
- performance reports,
- arquitectura detectada.

## Tipos de retos
- debugging,
- performance,
- refactorización,
- arquitectura,
- separación de dominio,
- optimización.

## Reglas
- usar componentes reales,
- evitar ejercicios artificiales,
- escalar complejidad progresivamente.
