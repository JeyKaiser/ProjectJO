# Session Generation Rules

## Objetivo
Generar automáticamente sesiones de estudio contextualizadas.

## Componentes posibles
- explicación técnica,
- debugging,
- refactorización,
- quiz,
- simulación,
- profiling,
- revisión arquitectónica.

## Reglas
- usar ejemplos reales del proyecto,
- conectar teoría con implementación,
- mezclar reasoning y práctica,
- evitar contenido genérico.

## Resultado esperado
Sesiones dinámicas y altamente contextuales.
