# Automatic Repo Analysis

## Objetivo
Analizar automáticamente el repositorio y detectar:
- estructura,
- complejidad,
- riesgos,
- escalabilidad,
- acoplamiento.

## Áreas observadas
- componentes,
- hooks,
- contexto,
- routing,
- CSS,
- dominio.

## Resultado esperado
Generación automática de hallazgos técnicos.
