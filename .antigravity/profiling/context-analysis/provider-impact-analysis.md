# Provider Impact Analysis

## Objetivo
Evaluar cómo un provider afecta el rendimiento global.

## Qué revisar
- tamaño del contexto,
- frecuencia de updates,
- cantidad de consumidores,
- granularidad del estado.

## Riesgos
- rerenders masivos,
- cascadas invisibles,
- dependencia excesiva.

## Objetivo pedagógico
Entender Context API desde performance engineering.
