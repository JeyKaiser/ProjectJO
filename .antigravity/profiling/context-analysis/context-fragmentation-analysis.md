# Context Fragmentation Analysis

## Objetivo
Evaluar si los contextos están correctamente fragmentados.

## Señales positivas
- providers pequeños,
- dominios separados,
- bajo impacto de actualización.

## Señales negativas
- provider único gigante,
- demasiados valores compartidos,
- updates globales frecuentes.

## Objetivo pedagógico
Aprender diseño granular de estado global.
