# Global State Render Analysis

## Objetivo
Analizar cómo el estado global propaga renders.

## Qué observar
- updates globales,
- renders en cascada,
- consumidores innecesarios,
- acoplamiento del estado.

## Riesgos
- baja escalabilidad,
- degradación progresiva,
- debugging complejo.

## Objetivo pedagógico
Comprender costo real del estado global.
