# Context Optimization Strategies

## Objetivo
Definir estrategias racionales para optimizar Context API.

## Estrategias
- fragmentar providers,
- mover estado localmente,
- memoizar values,
- separar dominio,
- reducir consumidores.

## Riesgos
- sobreingeniería,
- fragmentación excesiva,
- complejidad innecesaria.

## Objetivo pedagógico
Optimizar contexto sin destruir claridad arquitectónica.
