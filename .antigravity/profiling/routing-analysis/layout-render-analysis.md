# Layout Render Analysis

## Objetivo
Analizar cómo los layouts afectan el renderizado global.

## Qué evaluar
- layouts persistentes,
- renders innecesarios,
- providers dentro de layouts,
- composición de rutas.

## Riesgos
- rerenders amplificados,
- providers globales costosos,
- navegación pesada.

## Objetivo pedagógico
Comprender impacto arquitectónico de layouts.
