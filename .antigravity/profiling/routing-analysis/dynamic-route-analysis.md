# Dynamic Route Analysis

## Objetivo
Evaluar uso de rutas dinámicas y sincronización con estado.

## Qué revisar
- params,
- navegación programática,
- sincronización de datos,
- cambios de referencia.

## Riesgos
- stale state,
- render desincronizado,
- datos incorrectos.

## Objetivo pedagógico
Comprender interacción entre routing y lifecycle.
