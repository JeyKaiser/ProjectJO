# Route Performance Analysis

## Objetivo
Evaluar impacto del routing sobre performance y experiencia de navegación.

## Qué observar
- tiempo de carga,
- rerenders por navegación,
- layouts persistentes,
- sincronización de estado.

## Riesgos
- rutas pesadas,
- navegación inconsistente,
- renderización innecesaria.

## Objetivo pedagógico
Comprender navegación como flujo de renderizado.
