# Navigation Flow Analysis

## Objetivo
Analizar claridad y eficiencia del flujo de navegación.

## Evaluar
- predictibilidad,
- persistencia visual,
- profundidad de navegación,
- estado compartido entre vistas.

## Riesgos
- navegación confusa,
- pérdida de contexto,
- estados inconsistentes.

## Objetivo pedagógico
Pensar UX y arquitectura simultáneamente.
