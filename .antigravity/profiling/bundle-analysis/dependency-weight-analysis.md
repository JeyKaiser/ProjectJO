# Dependency Weight Analysis

## Objetivo
Detectar dependencias que impactan significativamente el tamaño del bundle.

## Qué evaluar
- librerías grandes,
- imports completos innecesarios,
- dependencias duplicadas,
- uso real vs peso agregado.

## Riesgos
- bundle inflado,
- carga lenta,
- deuda técnica invisible.

## Objetivo pedagógico
Desarrollar criterio sobre adopción de dependencias.
