# Lazy Loading Analysis

## Objetivo
Evaluar implementación de carga diferida.

## Qué revisar
- rutas diferidas,
- componentes pesados,
- fallback UX,
- Suspense boundaries.

## Riesgos
- bundle inicial excesivo,
- flashes visuales,
- navegación lenta.

## Buenas prácticas
- diferir páginas grandes,
- usar Suspense correctamente,
- evitar carga innecesaria temprana.

## Objetivo pedagógico
Aprender performance basada en distribución de código.
