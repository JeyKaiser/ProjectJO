# Route Splitting Analysis

## Objetivo
Analizar división del código basada en rutas.

## Qué evaluar
- rutas pesadas,
- separación de chunks,
- independencia entre vistas,
- carga inicial.

## Riesgos
- todo el sistema cargado desde inicio,
- navegación lenta,
- chunks demasiado grandes.

## Objetivo pedagógico
Comprender arquitectura moderna de bundles.
