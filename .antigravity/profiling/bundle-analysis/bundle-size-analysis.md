# Bundle Size Analysis

## Objetivo
Analizar tamaño y composición del bundle frontend.

## Qué evaluar
- dependencias pesadas,
- imports globales,
- vistas gigantes,
- assets innecesarios,
- duplicaciones.

## Riesgos
- carga lenta,
- navegación lenta,
- peor experiencia móvil.

## Señales positivas
- chunks pequeños,
- imports selectivos,
- división por rutas,
- lazy loading.

## Objetivo pedagógico
Comprender impacto del bundle en UX real.
