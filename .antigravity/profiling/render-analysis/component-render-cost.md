# Component Render Cost

## Objetivo
Evaluar el costo de renderización de un componente.

## Factores
- tamaño del JSX,
- cantidad de cálculos,
- listas renderizadas,
- hooks usados,
- dependencias,
- complejidad visual.

## Riesgos
- componentes pesados,
- render lento,
- actualizaciones costosas.

## Señales positivas
- componentes pequeños,
- cálculos memoizados correctamente,
- composición limpia.

## Objetivo pedagógico
Aprender a pensar en costo de render y no solo funcionalidad.
