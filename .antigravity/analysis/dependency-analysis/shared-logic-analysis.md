# Shared Logic Analysis

## Objetivo
Detectar lógica repetida o dispersa.

## Evaluar
- duplicación,
- validaciones repetidas,
- cálculos similares,
- patrones repetidos.

## Riesgos
- bugs duplicados,
- mantenimiento costoso,
- inconsistencia.

## Objetivo pedagógico
Aprender reutilización estratégica.
