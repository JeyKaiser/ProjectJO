# Coupling Analysis

## Objetivo
Detectar acoplamiento excesivo entre módulos.

## Evaluar
- dependencias fuertes,
- conocimiento excesivo entre capas,
- módulos difíciles de reutilizar.

## Riesgos
- cambios peligrosos,
- baja flexibilidad,
- escalabilidad limitada.

## Objetivo pedagógico
Aprender diseño desacoplado.
