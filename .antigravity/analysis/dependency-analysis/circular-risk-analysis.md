# Circular Risk Analysis

## Objetivo
Detectar riesgos de dependencias circulares.

## Evaluar
- imports cruzados,
- módulos dependientes entre sí,
- referencias mutuas.

## Riesgos
- comportamiento impredecible,
- debugging difícil,
- arquitectura frágil.

## Objetivo pedagógico
Comprender dependencias saludables.
