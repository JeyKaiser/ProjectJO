# UI Consistency Analysis

## Objetivo
Detectar inconsistencias visuales y semánticas.

## Evaluar
- spacing,
- tipografía,
- colores,
- tamaños,
- iconografía.

## Riesgos
- UX inconsistente,
- percepción amateur,
- baja mantenibilidad visual.

## Objetivo pedagógico
Desarrollar criterio visual sistemático.
