# CSS Monolith Analysis

## Objetivo
Detectar problemas de escalabilidad en CSS global.

## Evaluar
- tamaño,
- duplicación,
- selectores genéricos,
- dependencia global.

## Riesgos
- colisiones,
- miedo a refactorizar,
- !important excesivo.

## Objetivo pedagógico
Comprender arquitectura CSS mantenible.
