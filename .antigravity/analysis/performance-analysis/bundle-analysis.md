# Bundle Analysis

## Objetivo
Analizar tamaño, carga y división del bundle.

## Evaluar
- páginas pesadas,
- imports innecesarios,
- lazy loading,
- code splitting.

## Riesgos
- carga lenta,
- navegación pesada,
- bundle monolítico.

## Objetivo pedagógico
Comprender performance desde distribución de código.
