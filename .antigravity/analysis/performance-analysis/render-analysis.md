# Render Analysis

## Objetivo
Detectar renderizados innecesarios y comprender el lifecycle de React.

## Evaluar
- renders repetidos,
- props recreadas,
- callbacks recreados,
- listas grandes,
- cálculos costosos.

## Riesgos
- UI lenta,
- consumo innecesario,
- mala experiencia.

## Conceptos clave
- referencia por identidad,
- reconciliación,
- memoización,
- React.memo,
- useMemo,
- useCallback.

## Objetivo pedagógico
Pensar en performance desde arquitectura.
