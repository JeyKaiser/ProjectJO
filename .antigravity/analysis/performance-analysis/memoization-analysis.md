# Memoization Analysis

## Objetivo
Analizar uso correcto e incorrecto de memoización.

## Evaluar
- uso innecesario,
- ausencia donde aporta valor,
- dependencias incorrectas,
- complejidad agregada.

## Riesgos
- optimización prematura,
- bugs difíciles,
- complejidad innecesaria.

## Objetivo pedagógico
Aprender cuándo optimizar y cuándo no.
