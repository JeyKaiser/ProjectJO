# Context Performance Analysis

## Objetivo
Evaluar impacto del Context API en renderizado global.

## Evaluar
- tamaño del provider,
- frecuencia de cambios,
- renders en cascada,
- objetos recreados.

## Riesgos
- rerenders masivos,
- bajo aislamiento,
- performance degradada.

## Objetivo pedagógico
Comprender límites reales de Context API.
