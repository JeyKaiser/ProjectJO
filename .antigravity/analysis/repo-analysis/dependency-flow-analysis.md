# Dependency Flow Analysis

## Objetivo
Analizar cómo fluye la dependencia entre módulos y detectar acoplamientos peligrosos.

## Evaluar
- imports cruzados,
- dependencias implícitas,
- flujo de datos,
- reutilización,
- capas violadas.

## Riesgos
- dependencia circular,
- acoplamiento fuerte,
- módulos difíciles de aislar,
- flujo confuso.

## Señales positivas
- dependencias claras,
- flujo unidireccional,
- módulos independientes,
- responsabilidades aisladas.

## Objetivo pedagógico
Comprender cómo la dependencia afecta escalabilidad.
