# Folder Organization Analysis

## Objetivo
Evaluar si la estructura de carpetas representa correctamente el dominio y las responsabilidades del sistema.

## Aspectos a revisar
- agrupación funcional,
- separación por capas,
- consistencia,
- claridad de navegación,
- organización escalable.

## Riesgos
- carpetas genéricas,
- dependencia excesiva de utils,
- mezcla entre páginas y lógica,
- archivos sin contexto claro.

## Señales positivas
- estructura intuitiva,
- nombres coherentes,
- crecimiento predecible,
- modularidad.

## Objetivo pedagógico
Comprender cómo la estructura influye en mantenibilidad y onboarding.
