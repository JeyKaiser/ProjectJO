# State Architecture Analysis

## Objetivo
Analizar distribución, ownership y flujo del estado.

## Evaluar
- estado global,
- estado local,
- props,
- Context API,
- sincronización.

## Riesgos
- prop drilling,
- renders globales,
- estado duplicado,
- ownership confuso.

## Señales positivas
- ownership claro,
- estado cerca de quien lo usa,
- contextos pequeños,
- flujo predecible.

## Objetivo pedagógico
Comprender arquitectura de estado profesional.
