# Study Analytics — ProjectJO Learning System

## Objetivo
Detectar patrones de aprendizaje para priorizar temas y reforzar debilidades.

## Variables a observar
- temas estudiados con mayor frecuencia,
- temas con más errores,
- tiempo de comprensión,
- temas que generan confusión,
- respuesta ante ejercicios prácticos.

## Indicadores útiles
- Repetición alta con baja comprensión = tema crítico.
- Buen desempeño teórico pero mala aplicación = falta práctica.
- Respuesta correcta sin explicación clara = comprensión superficial.
- Dudas recurrentes = prioridad de refuerzo.

## Uso
Estas métricas deben alimentar el roadmap y la dificultad de las sesiones futuras.
