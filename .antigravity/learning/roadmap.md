# Roadmap — ProjectJO Learning System

## Nivel 1: Fundamentos y comprensión del proyecto
Objetivo:
Entender el stack, la estructura y el flujo general.

Temas:
- React 19
- Vite
- Router
- Context API
- estructura de carpetas
- dominio de temperaturas

## Nivel 2: Componentes y patrones
Objetivo:
Identificar responsabilidades, composición y separación de lógica.

Temas:
- componentes funcionales
- prop drilling
- composición
- custom hooks
- separación UI/dominio

## Nivel 3: Estado y rendimiento
Objetivo:
Comprender render lifecycle, memoización y optimización.

Temas:
- re-renderizados
- `useMemo`
- `useCallback`
- `React.memo`
- inmutabilidad
- performance profiling

## Nivel 4: Arquitectura CSS y UX
Objetivo:
Dominar mantenibilidad visual y consistencia.

Temas:
- CSS variables
- modularización
- design tokens
- responsive design
- glassmorphism
- escalabilidad visual

## Nivel 5: Senior Frontend Engineering
Objetivo:
Pensar como arquitecto senior.

Temas:
- decisiones de arquitectura
- escalabilidad
- refactorización
- evaluación de tradeoffs
- entrevistas técnicas
- diseño orientado al dominio

## Regla de avance
No avanzar de nivel hasta demostrar comprensión práctica del nivel anterior.
