# Performance Assessment

## Objetivo
Evaluar capacidad de identificar y resolver problemas de rendimiento.

## Evaluación
- detectar renders innecesarios,
- identificar mutaciones,
- comprender memoización,
- analizar Context API,
- optimizar listas y cálculos.

## Preguntas clave
- ¿Cuándo usar useMemo?
- ¿Cuándo NO usar memoización?
- ¿Qué provoca un re-render?
- ¿Qué problemas tiene Context API grande?

## Nivel esperado
Intermedio/Senior.
