# Weakness Detection Engine

## Objetivo
Detectar debilidades técnicas recurrentes.

## Señales observadas
- errores repetidos,
- confusión conceptual,
- debugging incorrecto,
- razonamiento incompleto,
- respuestas superficiales.

## Categorías
- React,
- performance,
- arquitectura,
- estado,
- CSS,
- debugging.

## Regla
Las debilidades deben registrarse y priorizarse automáticamente.
