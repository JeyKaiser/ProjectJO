# Mastery Detection

## Objetivo
Detectar dominio real de un tema.

## Un tema se considera dominado cuando:
- puede explicarse claramente,
- puede aplicarse,
- puede debuggearse,
- puede enseñarse,
- puede relacionarse con arquitectura.

## Señales falsas de dominio
- repetir definiciones,
- responder sin reasoning,
- depender de pistas constantes.

## Objetivo pedagógico
Diferenciar memoria de comprensión real.
