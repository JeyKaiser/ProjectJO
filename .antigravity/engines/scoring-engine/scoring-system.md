# Scoring System

## Objetivo
Asignar métricas técnicas dinámicas.

## Categorías
- React,
- arquitectura,
- debugging,
- performance,
- CSS,
- reasoning.

## Escala
0-100.

## Factores evaluados
- comprensión,
- aplicación,
- debugging,
- claridad,
- independencia,
- profundidad.

## Regla
El score debe reflejar criterio técnico, no memorización.
