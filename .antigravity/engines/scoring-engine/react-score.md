# React Score

## Evaluar
- hooks,
- render lifecycle,
- composición,
- estado,
- routing,
- effects.

## Indicadores senior
- separación lógica/UI,
- control de renders,
- composición limpia,
- razonamiento sobre estado.

## Penalizaciones
- mutación,
- prop drilling excesivo,
- lógica desordenada,
- effects incorrectos.
