# Architecture Score

## Evaluar
- separación de capas,
- modularidad,
- escalabilidad,
- dominio,
- mantenibilidad.

## Indicadores senior
- pensamiento sistémico,
- tradeoffs,
- desacoplamiento,
- claridad estructural.

## Penalizaciones
- acoplamiento,
- componentes monolíticos,
- lógica mezclada.
