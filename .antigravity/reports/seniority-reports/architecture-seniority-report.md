# Architecture Seniority Report

## Objetivo
Medir madurez arquitectónica del usuario.

## Criterios
- separación de capas
- modelado de dominio
- modularidad
- escalabilidad
- mantenimiento
- capacidad de detectar acoplamiento

## Niveles
### Básico
- reconoce componentes y carpetas

### Intermedio
- entiende responsabilidades
- identifica problemas de organización

### Avanzado
- propone refactors
- detecta acoplamientos
- diseña capas

### Senior
- modela sistemas
- evalúa tradeoffs
- diseña para crecimiento

## Resultado esperado
- madurez actual
- puntos débiles
- criterios por fortalecer
- próximos temas de arquitectura
