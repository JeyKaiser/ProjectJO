# Frontend Seniority Report

## Objetivo
Evaluar el nivel de madurez frontend del usuario.

## Criterios
- comprensión de hooks
- manejo de estado
- conocimiento de routing
- composición de componentes
- criterio de performance
- claridad visual
- capacidad de refactorización

## Niveles
### Básico
- uso correcto de componentes y props

### Intermedio
- manejo de estado y efectos
- comprensión de routing
- refactorizaciones simples

### Avanzado
- control de renderizado
- composición limpia
- patrones de reutilización

### Senior
- decisiones arquitectónicas
- tradeoffs técnicos
- diseño escalable
- mantenimiento a largo plazo

## Resultado esperado
- nivel actual
- brechas de seniority
- fortalezas
- siguiente escalón
