# Performance Seniority Report

## Objetivo
Evaluar la madurez del usuario en performance frontend.

## Criterios
- comprensión del render lifecycle
- detección de renders innecesarios
- uso correcto de memoización
- criterio para optimizar
- comprensión de bundle y carga
- evaluación del impacto real

## Niveles
### Básico
- sabe que existe performance

### Intermedio
- identifica algunos problemas evidentes

### Avanzado
- propone optimizaciones con sentido

### Senior
- mide antes de optimizar
- reconoce tradeoffs
- evita sobreingeniería
- piensa en usuario y costo técnico

## Resultado esperado
- nivel actual
- oportunidades de mejora
- errores típicos
- próximo objetivo
