# Weakness Evolution Report

## Objetivo
Registrar cómo cambian y se reducen las debilidades técnicas con el tiempo.

## Qué observar
- errores repetidos
- confusiones persistentes
- temas superados
- nuevas debilidades emergentes
- estabilidad conceptual

## Uso
Este reporte sirve para ajustar el nivel de dificultad y priorizar entrenamiento.

## Resultado esperado
- debilidades activas
- debilidades en retroceso
- debilidades críticas
- foco siguiente de práctica
