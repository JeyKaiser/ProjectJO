# Frontend Audit Template

## Objetivo
Generar una auditoría general del frontend enfocada en estructura, calidad de código, arquitectura y experiencia de mantenimiento.

## Alcance
- componentes
- páginas
- hooks
- routing
- estado global
- estilos
- dominio de negocio
- consistencia visual

## Criterios de evaluación
### 1. Estructura
- organización del código
- claridad de carpetas
- separación por responsabilidades

### 2. Calidad de implementación
- legibilidad
- complejidad
- reutilización
- consistencia

### 3. Arquitectura
- acoplamiento
- modularidad
- escalabilidad
- uso correcto de capas

### 4. UX técnica
- flujo de navegación
- consistencia visual
- carga de vistas
- claridad funcional

### 5. Riesgos
- código duplicado
- componentes monolíticos
- estado mal distribuido
- CSS global inmanejable

## Hallazgos esperados
- fortalezas
- debilidades
- riesgos prioritarios
- oportunidades de refactor

## Resultado final
La auditoría debe terminar con:
1. resumen ejecutivo
2. hallazgos críticos
3. hallazgos medios
4. hallazgos menores
5. plan de acción recomendado
