# Maintainability Audit Template

## Objetivo
Medir qué tan fácil es mantener, entender y extender el proyecto.

## Aspectos evaluados
- legibilidad
- tamaño de archivos
- composición
- repetición de lógica
- consistencia de patrones
- claridad de nombres
- separación UI/dominio

## Señales de buena mantenibilidad
- responsabilidades pequeñas y claras
- hooks reutilizables
- componentes enfocados
- CSS ordenado
- datos y lógica bien separados

## Señales de baja mantenibilidad
- componentes gigantes
- estilos globales frágiles
- lógica distribuida sin criterio
- estructuras confusas
- abundancia de excepciones manuales

## Salida esperada
- diagnóstico de mantenibilidad
- deuda técnica visible
- recomendaciones prácticas
- prioridades de limpieza
